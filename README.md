# CookMyBots WhatsApp Bot (Managed WhatsApp)

This WhatsApp bot is generated by **CookMyBots**.

## Important
- **Do NOT** add WhatsApp Cloud API tokens here.
- WhatsApp transport is handled by CookMyBots after you connect your number in the dashboard using Session QR mode or Cloud API mode.

## What runs in this repo?
This repo hosts the bot "brain" (AI + optional storage).

CookMyBots owns the WhatsApp connection/session and forwards inbound WhatsApp messages to:

POST /webhook/cookmybots/whatsapp

That means each deployed WhatsApp bot can behave independently while CookMyBots handles WhatsApp transport.
## Run
- `npm install`
- `cp .env.sample .env` and fill values
- `npm run dev`

## Local test
POST http://localhost:3000/test
Body:
{ "text": "hello" }

## CookMyBots transport endpoint
POST /webhook/cookmybots/whatsapp

Headers:
X-CookMyBots-Webhook-Secret: your deployed secret

Body:
{
  "from": "2348012345678@s.whatsapp.net",
  "text": "hello",
  "messageId": "message-id"
}
